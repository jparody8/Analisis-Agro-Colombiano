# Una Mirada al Campo Colombiano: Análisis de Productividad Agrícola🌾

Este proyecto realiza un Análisis Exploratorio de Datos (EDA) detallado sobre la productividad del sector agropecuario en Colombia, utilizando como base los datos de la Evaluación Agropecuaria Municipal (EVA). El objetivo es identificar patrones geográficos, temporales y por tipo de cultivo para entender la dinámica del agro en el país.

## 🎯 Objetivos del Proyecto
* **Entendimiento del Negocio:** Analizar cómo los beneficiarios (entidades gubernamentales y gremios) pueden utilizar estos datos para mejorar la planificación rural y el diseño de políticas públicas.
* **Análisis Estadístico:** Evaluar el desbalanceo en las categorías de crecimiento, grupos de cultivos y distribución geográfica.
* **Visualización de Datos:** Crear reportes analíticos que faciliten la interpretación de tendencias entre los años 2023 y 2024.

## 🛠️ Tecnologías Utilizadas
* **Python:** Lenguaje principal para el procesamiento de datos.
* **Pandas & NumPy:** Manipulación y limpieza de grandes conjuntos de datos.
* **Matplotlib & Seaborn:** Generación de gráficos estadísticos y visualizaciones geográficas.
* **Google Colab:** Entorno de desarrollo para la ejecución de cuadernos interactivos.

## 📊 Hallazgos Principales
* Se identificó un desbalanceo moderado en las categorías de crecimiento y uno significativo en los grupos de cultivos (predominio de cereales y frutales), lo cual refleja la realidad productiva del país.
* La distribución temporal de los datos se mantiene balanceada entre los periodos analizados, permitiendo comparaciones anuales sólidas.
